자재마스터 자동 수정 도구 서식유지

엑셀 두 파일을 업로드하여 MX 코드 매칭 · spec 제거 · MAKER P/N 추출을 자동 수행합니다. (셀 서식 보존)

1파일 업로드

2첨부1 (수정 대상) 설정

파일을 업로드하면 시트 목록이 자동으로 채워집니다. 컬럼은 알파벳(A, B, C…)으로 입력하세요.

3처리 옵션

3/8", DR 6각, 1", 4CH, E/P, M6, 220V 등 부가 사양을 자재명에서 제거합니다.
+ 사용자 정의 제거 키워드 (선택사항)
기본 spec 제거 후 추가로 적용됩니다. 대소문자 무시 · 단어 경계 자동 인식. 정규식 모드를 켜면 각 항목을 정규식으로 처리합니다 (예: \bTEST\d+\b).

4첨부2 매핑 설정

첨부2의 어느 시트·컬럼에서 데이터를 가져올지 지정합니다. 시트를 선택하면 위례 자재마스터 표준에 맞춰 컬럼이 자동으로 채워집니다. 다른 사업소 파일의 경우 자동으로 채워진 값을 직접 수정해 사용하세요.
제조사 강제값이 입력된 시트는 제조사 컬럼을 무시하고 그 값(예: SIEMENS)을 J열에 기입합니다. 비워두면 컬럼에서 읽습니다.
Siemens Parts가 포함된 시트에는 Revision P/N 옵션이 자동으로 표시됩니다.

5결과 파일 저장

미리 폴더를 선택하지 않은 경우, '수행'을 누르면 저장 위치를 묻는 대화상자가 먼저 뜹니다.
